# Data Jobs Intelligence

Projeto de inteligência de mercado de trabalho desenvolvido com Python, SQL Server e Power BI.

## Objetivo

Coletar automaticamente vagas publicadas em plataformas de recrutamento, estruturar os dados e gerar análises sobre:

* Tecnologias mais demandadas
* Distribuição geográfica das vagas
* Modalidade de trabalho
* Tendências de contratação
* Habilidades requisitadas pelo mercado

## Tecnologias

* Python
* Requests
* BeautifulSoup
* Pandas
* SQL Server
* Power BI
* Git
* GitHub
